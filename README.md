# Tui-textbox

Stateful widget for [ratatui](https://github.com/ratatui-org/ratatui).

## Example

Check `example/`.

To run demo:

```shell
cargo run --example demo
```

![demo](./tui-textbox-demo.gif)

## Features

- Stateful
- Support moving cursor `Left` and `Right`
- `Backspace` and `Delete` supported
- Paste from clipboard
